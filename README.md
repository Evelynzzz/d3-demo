# d3-demo
demos with d3.js 
 - react
 - webpack 2
 - hot reload
 - babel
 - extract-text-webpack-plugin

# Use
```
npm install
```
- Development mode

    `node server.js` or `npm start`

    Open http://localhost:3000/
- Production mode
    
    `sh ./pack_run.sh` or `npm run build`

    Open *index.html* in the browser directly.
